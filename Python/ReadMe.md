# 파이썬의 메모리 구조

# 변수
1. 리스트와 튜플
- 리스트 : [] 선언
- 튜플 : () 선언
  - 한개의 값 : tuple1 = (1,)
  - 괄호 생략 가능 : tuple2 = 1,2,3,4
- 리스트의 값을 생성, 삭제, 수정이 가능
- 튜플은 값 변경 불가.


2. 집합 자료형
- 값의 중복을 허용하지않는다.
- 순서가 없다.

따라서 값의 순서가 없기 때문에 리스트,튜플로 변경하여 인덱싱을 해야한다.
```python
s1 = set([1,2,3])
l1 = list(s1)
```

# 클래스 Class
- 메서드 선언시 첫번째 매개변수는 self로 명시적으로 구현하는것이 관례이다.
## 생성자
- 클래스 내에서 메서드 이름으로 __init__ 를 사용하면 메서드 생성자가 된다.
